# NAME

OpenIMP::APIClient - Perl client for the OpenIMP API

# SYNOPSIS

  use OpenIMP::APIClient;

  ...

# DESCRIPTION

Wraps around the OpenIMP API to provide a Perl object interface to the OpenIMP
API.
